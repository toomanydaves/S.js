view.Viewer Class

Defined in: js/ls/view/Viewer.js:8
Module: view

Viewer is a solution for communicating the user's current context within the application. Context in this sense refers to a set of related viewpoints. Each viewpoint has a name recognizable to the user and a unique reference to its contents.

Viewer uses a Viewport and a Path instance for display purposes. The path shows the name and order of viewpoints in the current context; the viewport displays the contents of the viewpoints.
