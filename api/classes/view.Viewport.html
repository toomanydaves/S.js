view.Viewport Class

Defined in: js/ls/view/Viewport.js:5
Module: view

Viewport is used to display the screens holding the content of individual viewpoints. These screens can be shifted left and right, as the user moves deeper or shallower in a particular context, or replaced entirely when the context changes.
